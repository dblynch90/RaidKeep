World of Warcraft Classic Game Data APIs

The World of Warcraft Classic game data APIs encompass both static and dynamic game data. For more information about static and dynamic game data, see World of Warcraft Classic Namespaces.

Host Names

For all regions except China, the API host name follows the format {region}.api.blizzard.com, where {region} is one of us | eu | kr | tw. For China, the host name is gateway.battlenet.com.cn. For more information, see Regionality and APIs.

Namespaces

The World of Warcraft Classic game data APIs require a namespace as part of every request. Each endpoint below lists the correct namespace to use as a parameter. A namespace can be submitted as a header (Battlenet-Namespace: {namespace}) or as a query parameter (?namespace={namespace}). For more information, see World of Warcraft Classic Namespaces.

APIs

World of Warcraft Classic supports the following game data APIs. For information about using the APIs, see Getting Started.

 Auction House API
GET
Auction House Index
/data/wow/connected-realm/{connectedRealmId}/auctions/index

Returns an index of auction houses for a connected realm.

See the Connected Realm API for information about retrieving a list of connected realm IDs.

ParameterType Value Description
:region string
Required

The region of the data to retrieve.

{connectedRealmId} integer
Required

The ID of the connected realm.

namespace string
Required

The namespace to use to locate this document.

locale string

The locale to reflect in localized data.

GET
Auctions
/data/wow/connected-realm/{connectedRealmId}/auctions/{auctionHouseId}

Returns all active auctions for a specific auction house on a connected realm.

See the Connected Realm API for information about retrieving a list of connected realm IDs.

Auction house data updates at a set interval. The value was initially set at 1 hour; however, it might change over time without notice.

Depending on the number of active auctions on the specified connected realm, the response from this endpoint may be rather large, sometimes exceeding 10 MB.

ParameterType Value Description
:region string
Required

The region of the data to retrieve.

{connectedRealmId} integer
Required

The ID of the connected realm.

{auctionHouseId} integer
Required

The ID of the auction house.

namespace string
Required

The namespace to use to locate this document.

locale string

The locale to reflect in localized data.
